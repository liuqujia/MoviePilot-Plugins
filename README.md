# MoviePilot 插件仓库

## ASSRT字幕下载

整理入库时自动从 ASSRT 搜索并下载字幕。

### 功能

- 自动搜索匹配的字幕
- 支持简体/繁体/英文字幕
- 自动下载并保存到视频目录
- 支持覆盖已有字幕

### 配置

1. 获取 ASSRT API Token
   - 注册 https://assrt.net
   - 在个人设置中获取 API Token

2. 在 MoviePilot 中添加仓库
   ```
   https://github.com/liuqujia/MoviePilot-Plugins
   ```

3. 安装插件并配置 Token
